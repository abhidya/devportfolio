UT Maps

My website I built solves the problem of knowing where UT campus printers are and what they are named. The website centers a google map on the users' location and loads custom markers on where every live printer is allowing the user to find the closest printer. Clicking on the icon renders more information about the printers name, make and model. It also shows information about if the printer can print color or black and white.

Software for Bio-Informatics

A web interface to allow data collection of Plasmodium parasite concentration in mosquito species. Developed a prediction model based on the results in “How Plasmodium parasites cause malaria: understanding transmission of parasites from mosquitos to mice”. Model calculates the probability of a Malaria infection if bitten by a mosquito. This was developed during an independent research course advised by Dr. Michael W. Berry & Dr. Vitaly Ganusov.
